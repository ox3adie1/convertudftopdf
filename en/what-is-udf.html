What is UDF? The Turkish UYAP Document Format Explained

Last updated: April 7, 2026 · 6 min read

If you've received a file with a .udf extension from a Turkish lawyer, court, or legal professional, you're likely wondering what it is and how to open it. This guide explains everything you need to know about the UDF file format and Turkey's UYAP system.

UDF: UYAP Document Format

UDF stands for UYAP Document Format. It is a proprietary document format created by and for Turkey's national judiciary informatics system. Every legal document produced within the UYAP Document Editor — petitions, court decisions, hearing minutes, indictments, notifications — is saved in this format.

In brief: UDF is to the Turkish legal system what .docx is to Microsoft Word — except it can only be opened by one specific application that's restricted to the UYAP network.

What is UYAP?

UYAP (Ulusal Yargi Agi Platformu) — the National Judiciary Informatics System — is Turkey's comprehensive digital judiciary platform. Developed by the Ministry of Justice starting in the early 2000s, it is one of the most extensive e-justice systems in the world.

UYAP by the Numbers

  • 200,000+ active users (judges, prosecutors, lawyers, court staff)
  • 85+ million citizens with access through e-Government portal
  • 1,000+ connected courthouses and institutions
  • 20+ years of operation and development

What UYAP Does

  • Case management: All cases from filing to resolution are managed digitally
  • Document creation: The UYAP Document Editor creates legal documents in UDF format
  • Electronic signatures: All official documents are signed with qualified e-signatures
  • Lawyer portal: Lawyers can file petitions, track cases, and receive notifications online
  • Citizen portal: Citizens can check case status and access court information
  • Video hearings (SEGBIS): Remote court hearings via video conferencing
  • Electronic notifications: Legal notifications delivered digitally

Technical Details of UDF Files

UDF files have the following characteristics:

  • File extension: .udf
  • Internal structure: ZIP-compressed archive containing document data
  • Content format: Rich text with formatting (bold, italic, underline, fonts, alignment)
  • Table support: Legal documents frequently include structured tables
  • Character set: Full Turkish character support (including special characters like ş, ğ, ı, ö, ü, ç)
  • Template system: Pre-built templates for petitions, decisions, minutes, etc.

How to Open a UDF File

Option 1: UYAP Document Editor (Official)

The official way to open UDF files is through the UYAP Document Editor. However, this application is only available on computers connected to the UYAP network — typically found in courthouses, prosecutors' offices, and law firms with UYAP access.

UDF vs PDF Comparison

  • Compatibility: PDF works on every device and operating system. UDF only works with the UYAP Document Editor.
  • Mobile support: PDF has full mobile support. UDF has none.
  • Browser support: PDF can be viewed in any web browser. UDF cannot.
  • Printing: PDF can be printed from any printer. UDF requires the UYAP Editor.
  • Email sharing: PDF recipients can always open the file. UDF recipients may not be able to.
  • Archiving: PDF/A is an international archiving standard. UDF is tied to the UYAP ecosystem.

Is the UDF format the same as ISO UDF (Universal Disk Format)?

No. Despite sharing the same abbreviation, UYAP's UDF (UYAP Document Format) and ISO 13346 UDF (Universal Disk Format, used for DVDs and Blu-rays) are completely different formats with no relation to each other.
